My Game development experiences

My main game development journey started two years ago, but I had some experience with game development before my time at university. At university, my main focus was coding games since I was already proficient at that, and I tried to learn as much as possible about game design, specifically system and content design. Before Uni, I created some smaller 2d platformers, and for my diploma thesis, I coded a VR Simulation of a message transmission between two phones.

Gameplay Coding

Over the last two years, I have developed multiple games from the ground up, coding everything from custom 2D movement and colliders to a modular ability creation system that non-coders can use.

System Design

I provided input on design decisions for all my games, contributing to the design team for mechanics and systems where I helped create the game's overarching mechanic set.

Idea Finding

I was part of the idea-finding process for every game I worked on, collaborating with multiple people to create exciting ideas and mechanics for new games.

More Coding Experience

Before studying game design, I attended one of Australia's top IT high schools, where I gained five years of experience learning multiple coding languages and standard software engineering best practices.

Coding and Design

I have been coding for seven years and have two years of experience in game design. My combined skills in coding and design allow me to approach technical problems with design in mind and vice versa. I am also flexible and can switch between coding and design tasks to help where I am needed most.

Start fast and adapted

When I develop games, I enjoy using this approach. Rapid prototyping and efficient iterating, testing, and refining ideas have always been better than theory-crafting for ages.

Communication

I really enjoy communicating while working. It's great to have someone to bounce ideas off for game mechanics or discuss how to implement them into the code. It keeps things flowing smoothly and makes the work better.
